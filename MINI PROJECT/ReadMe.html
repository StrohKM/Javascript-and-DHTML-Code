Mini Me Social Media About Page

This project I wanted to use javascript to allow the user to customize some different features.
Using javascript, jQuery, and DOM elements to create image uploads, image resizing, and draggable comment boxes and resizing comment boxes
I learned a lot in this project. I have never fully designed a webpage myself so this was very difficult to teach myself new things. I used lots of outside resources to help correct code that wasn't working and researched many different jQuery related things. I wish I had a bit more time to make this more polished.

This webpage allows you to do a couple things:

  • You can upload images, resize them (using the small gray triangle in the bottom right of the image), and drag them around
  • When you upload an image there is a frame that comes with it that can be resized to fit your image! Look for that second draggable corner
  • There is a banner at the top of the webpage where you can add information into
  • Feel free to change the themes using the drop down menu
  • If there is no longer a photo you want you can always delete it using the delete button
  • You are able to add comments using the 'add comment' button to your photos
  • Make sure you delete the comment before you start typing in it if you no longer want it
